# Log Linter

линтер Go, проверяющий лог-записи на соответствие установленным правилам. Совместим с golangci-lint и поддерживает популярные логгеры: log/slog и go.uber.org/zap.
## Возможности
Линтер проверяет, что:

- Сообщения начинаются со строчной буквы.

-  Сообщения только на английском языке, без спецсимволов и эмодзи.

- Сообщения не содержат потенциально чувствительные данные (password, token, api_key и т.д).

## Поддерживаемые логгеры

- `log/slog`
- `go.uber.org/zap`

## Установка и запуск

Клонируйте репозиторий:

``` git clone https://github.com/gegxkss/linter.git
cd linter 
```

Установите зависимости::
```
go mod tidy
```
## Для использования плагина убедитесь, что golangci-lint установлен:

Запуск линтера:
```
golangci-lint run 
```

### Линтер проверит лог-сообщения (log, slog, zap) на соответствие правилам, указанным в линтере, и покажет SuggestedFixes.

## Тестирование

В проекте есть тесты для проверки всех правил

Запуск тестов:
```
go test ./pkg/check

```
